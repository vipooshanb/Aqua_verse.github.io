Home Page

Technical Description

The webpage employs a responsive design with a marine theme, using a blue palette to reflect its ocean conservation focus. The structure includes:

  • A header with navigation links (Home, Team, Table, Volunteer, Site Map, Feedback, Profile) and a logo.

  • A main content area with card-based sections (Underwater Experience Places, Exploring Life Below Water, Threats Facing Our Oceans, Guardians of the Aquaverse, Building a Better Ocean Future).

  • A page editor section enabling multiple users (Students 1-4) to edit specific pages (Home, Future, Table, Additional).
The design uses HTML and CSS, likely with a framework for layout consistency, and includes interactive elements like "View Table" buttons.

Accessibility

Alt attributes for images, proper heading hierarchy, and high-contrast text. Responsive design ensures usability on all devices.

Content Page: Future of Our Blue Planet

Technical Description

Built with semantic HTML5 tags for accessibility and SEO. CSS Grid and Flex layouts for responsiveness. Aqua gradients, Pacifico font for branding, images, and animated bubbles for interactivity.

Accessibility

High-contrast text improves readability for visually impaired users.
Clear hierarchical structure with headings and sections supports screen reader navigation.
Consistent navigation layout aids cognitive accessibility.
